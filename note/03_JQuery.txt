# JQuery

	- 자바스크립트의 긴 문법들을 짧게 줄여놓은 라이브러리
	- document.querySelector("#id") -->> $("#id")
	- btn1.addEventListener("click", () => {}) -->> button1.on("click", () => {})
	- 문법의 길이를 줄이는 것 뿐만 아니라 자주 쓰이는 기능들을 편리하게 묶어놓은 경우도 있다.
	
# JQuery 다운로드 센터에 있는 파일 종류의 의미

	- uncompressed : 들여쓰기 규칙 잘 지킨 형태의 코드 (용량 큼, 보기 좋음)
	- minified : 용량을 줄이기 위해서 붙여쓴 형태의 코드
	- slim : 자주 쓰는 기능만 남겨놓은 버전
	- slim minified : 자주 쓰는 기능만 남겨놓은 버전에서 공백 및 들여쓰기도 제거한 버전
	
# JQuery에서 Ajax 요청하기
		
	- 다음 정보들을 담고 있는 자바스크립트 객체를 하나 생성한다.
		
		url : 요청 주소
		method : 요청 방식
		dataType : 요청 성공 후 받게 되는 데이터 타입
		success : 요청 성공 시 처리 함수
		statusCode : 해당 상태 코드가 도착했을 때의 처리들을 객체 형태로 정의
	
	- $.ajax()를 사용해 해당 객체와 함께 요청을 보낸다.
	
	- 요청시 콜백에서 받게 되는 데이터
	
		첫 번째 인자 : 응답 데이터
		두 번째 인자 : 응답 상태 코드
		세 번째 인자 : ajax 요청 객체
