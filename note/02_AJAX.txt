# AJAX(Async Javascript and XML)

	- 비동기 자바스크립트와 XML(또는 JSON)
	- 웹 페이지를 새로고침 하지 않고 요청을 보내 데이터를 응답 받아오는 자바스크립트의 문법
	- 주로 @RestController에 요청을 보내 데이터를 받아온 후 자바스크립트에서 활용하는 방식으로 사용된다.
	- 서버로부터 비동기 요청에 대한 결과를 응답받을 때마다 이벤트가 발생하고 해당 이벤트를 활용하여 로직을 구성할 수 있다.
	
# XMLHttpResquest

	- 자바스크립트의 비동기 요청 객체
	- 요청을 보내도 브라우저가 응답받은 페이지로 새로고침 되지 않는다.
	- 인스턴스를 생성한 후 여러가지 값을 설정한 뒤에 요청을 보낼 수 있다.
	
# xhttp의 onreadystatechange 이벤트

	- xhttp 객체 생성 이후부터 해당 객체의 상태가 변화할 때마다 발생하는 이벤트
	- readyState 1 : open()이 성공했을 때 (send() 하기 전)
	- readyState 2 : 요청에 대한 응답이 도착했을 때
	- readyState 3 : 응답에 대한 처리를 시작했을 때
	- readyState 4 : 응답에 대한 처리가 모두 완료되었을 때
	
# xhttp.open(method, uri)

	- 첫 번째 파라미터에는 요청 방식을 설정한다.(GET, POST, DELETE, PUT, PATCH, ...)
	- 두 번재 파라미터에는 요청 보낼 URI를 설정한다.

# xhttp.send(parameter)

	- 설정을 모두 마친 후 서버로 비동기 요청을 보내는 메서드
	- GET 방식을 제외한 나머지 요청에는 send() 안에 파라미터를 실어보낼 수 있다.(GET 방식은 주소 뒤에 ?를 통해 보낸다.)
	- 파라미터는 문자열로만 전송할 수 있다.
	- 파라미터 문자열을 폼 형식(name=value&name=value&..)으로 전달할 때는 요청 헤더의 content-type을 application/x-www-form-urlencoded로 설정해야 한다.
	- 파라미터 문자열을 JSON 형식으로 전달할 때는 요청 헤더의 content-type을 application/json으로 설정해야 한다.
